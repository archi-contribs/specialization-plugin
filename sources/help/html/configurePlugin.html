Specialization plugin

Configure the plugin

The plugin can be configured through its preference page that can be accessed using Archi's menu Edit / Preferences / specialization plugin.

The "Behaviour" tab allows to change the plugin's behaviour:

  • Version
The plugin's version is printed.
The "check for update" button allows to check if a new version is available on Github and automatically download it.
This check can also be ran automatically when Archi starts.
  • Behaviour
Please choose whether you with to replace the elements icons in the model tree.
You may also replace the elements icons on all the views, or only those having a property "replace icons" with value "true".
As well, you may also replace the elements and relationships labels on all the views, or only those having a property "replace labels" with value "true".
  • Online help
You may access the plugin help pages using this button, or using Archi's help menu.



The "Logger" tab allows to log the plugin's activity:

  • Disabled
In this mode, the plugin does not log any activity nor error.
  • Simple mode
In this mode, you only need to specify a log file and the severity of the messages.
  • Expert mode
This mode is for expert people as you will have access to the Log4J configuration.
