Specialization plugin

Plugin objectives

  • Executive summary
The Specialization plugin extends the Archi views with the following functionalities :
  • Allow to customize the elements icons
  • Allow to customize the elements and relationships labels

  • How to use the plugin
  1. Configure the plugin on the preference page

    • Logger
  2. Specialize your model
  3. Specialize your views
  4. Specialize your folders
  5. Replace icons
  6. Replace labels
