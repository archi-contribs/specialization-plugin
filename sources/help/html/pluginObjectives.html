Specialization plugin

Plugin objectives

  • Executive summary
The Specialization plugin extends the Archi views with the following functionalities :
  • Allow to customize the elements icons
  • Allow to customize the elements and relationships labels

  • How it works
  • Customize the elements icons
Icons are image files (png, jpg, gif, ...) in Archi's editor folder. this folder is "/plugins/com.archimatetool.editor_4.x.x.xxxxxxxxxxxx/img" under Archi's installation folder (where 'x' is dependent on archi's version)."
Please pay attention to not modify Archi's images. I recommand you create your own subfolders before copying your image files.

When this is done, you may create a "icon" property to your elements, the value being the path of your image file, relative to Archi's editor folder.
For instance, specifying "/os/windows.png" will refer to the image file "Archi's installation path/plugins/com.archimatetool.editor_4.x.x.xxxxxxxxxxxx/img/os/windows.png"


  • Customize the elements and relationships labels
You may create a "label" property to your elements or relationships, the value being label you wish to show in the views.
Setting or changing a label this way does not automatically update the view. To see your changes, you may right click on the view background and select "refresh view" in the contect menu that shows up.
