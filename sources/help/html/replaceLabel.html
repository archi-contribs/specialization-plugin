Specialization plugin

Replace label

When an element or relationship is selected in a view, a new "Replace label" tab is shown in the properties window.
  • Label
    • The text field allows to enter a label that will replace the name of the component in the view.
    • An empty value keeps the component's name
    • you may specify a "space" to hide the component's label
It is also possible to set the properties by hand. if you experience a "you must configure the view or the model to allow labels replacement" message, this means that you view does not allow the labels replacement.

You may change one of the following:
  • Change the "Replace labels in views" to "always" on the preference page
  • Change the "Replace labels in all views" in the model's properties
  • Change the "Replace labels in this view" in the view's properties
